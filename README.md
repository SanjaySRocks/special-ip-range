
# Special IP Address

Get Special IP Address range using hosting api


# Hosting Support
- AWS Lightsail
- AWS EC2 (coming soon)
- Digital Ocean


## Deployment

To deploy this project run

Install requirements
```bash
  pip install -r requirements
```

Run Script
```bash
  python aws.py
```
