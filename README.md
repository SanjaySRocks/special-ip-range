## Deployment

To deploy this project run

For Windows Activate Env
```bash
  .\env\Scripts\activate
```

For Linux Activate Env
```bash
  source env/bin/activate
```

Install requirements
```bash
  pip install -r requirements
```

Run Script
```bash
  py aws.py
```
